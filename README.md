# TomoChain Documentation

Docs for [TomoChain](https://docs.tomochain.com)

## Installation

Requires `python`, `pip`, `virtualenv`

Create working enviroment
```
virtualenv env
. env/bin/activate

```

Intall libraries
```
pip install -r requirements.txt
```

### Run
On linux, using bash script
```
bash docs.sh
```
